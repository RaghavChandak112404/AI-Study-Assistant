# AI Study Assistant

AI Study Assistant is a web-based platform that helps students learn more efficiently from PDF documents using NLP and AI.

The system extracts text from uploaded PDFs and uses Gemini AI to generate summaries and structured notes. It also provides an AI-powered chatbot that allows users to ask questions directly related to the uploaded document.

In addition, the platform includes a course recommendation feature that integrates Google Search API along with Coursera, Udemy, and YouTube APIs to suggest relevant learning resources based on the content of the uploaded PDF.

## Key Features
- PDF text extraction using NLP
- AI-based summarization and notes generation (Gemini AI)
- Document-based AI chatbot
- Course and resource recommendations
- Integration with Google, Coursera, Udemy, and YouTube APIs
